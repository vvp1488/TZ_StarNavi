# TZ_StarNavi
Реализовано API,из JWT-токен авторизацией,имеються endpoints(с правами доступа для разных действий):
-api/v1/users/ - GET, POST для просмотра списка пользователей и создание нового пользователя.
-api/v1/users/<pk>/ - GET,PUT,DELETE(редактировать или удалять пользователя может только сам пользователь или админ)
-api/token/ - POST запрос(для аунтефикации и создания токена для доступа)
-api/v1/posts/ - GET,POST запрос(добавлять запись может только авторизованый пользователь)
-api/v1/posts/<pk>/ - GET, PUT,DELETE (редактировать или удалять запись может создатель или админ)
-api/v1/posts/posts_current_user/ - GET, просмотр постов текущего пользователя
-api/v1/posts/<pk>/like/ -POST запрос для добавления лайка к записи
-api/v1/posts/<pk>/unlike/- POST пользователь убирает свой лайк
-api/v1/posts/<pk>/fans/ - GET что бы посмотреть кто лайкал запись
-api/v1/analitics/ - GET запрос из применением фильтра по дате лайка (?created_ad_after=2021-09-01&created_ad_before=2021-09-07) и общее количество лайков включая фильтры
-api/v1/user_activity/ - GET запрос(показывает статистику юзеров: дату последнего входа в систему(авторизацию) и дату и тип последнего запроса юзера к нашему сервису)

Инструмент для тестирования запросов : Postman
